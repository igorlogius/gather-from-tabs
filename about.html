Gather information from multiple tabs, by using a small snippet of javascript Usage
  1. Open the option page and create a fitting action (jaascript snippet)
  2. select / highlight the tabs you want to run the snippet on
  3. right click on any of the selected tabs to open the context menu
  4. in the context menu , select the Action you want to run from the 'Gather from Tabs' subentry
  5. Use CTRL+V to past the gathered output into any programm you like
